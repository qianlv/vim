# vim
vim配置, 使用[vunble](https://github.com/VundleVim/Vundle.vim)进行插件管理
    
## 配置

    cd ~/.vim  
  
if Git versions > 1.6.5  

    git clone --recursive git@github.com:qianlv/vim ./  

older Git versions  

    git clone git@github.com:qianlv/vim ./  
    git submodule update --init --recursive  
  
    cp ~/.vimrc vimrc.ba  
    cp vimrc ~/.vimrc
